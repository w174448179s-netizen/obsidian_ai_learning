# 面试学习知识点（高级Java后端工程师）

## 一、Java核心基础

### 1.1 面向对象
- 三大特性：封装、继承、多态
- 接口 vs 抽象类的区别
- 设计模式：单例模式、工厂模式、策略模式、模板方法模式、观察者模式、代理模式（动态代理）

### 1.2 多线程并发
- 线程创建方式：继承Thread、实现Runnable、Callable+Future
- 线程状态：新建、就绪、运行、阻塞、死亡
- synchronized关键字：对象锁、类锁、可重入性
- volatile关键字：可见性、禁止指令重排序、不保证原子性
- 并发工具类：
  - ReentrantLock：可重入锁、公平锁/非公平锁、Condition条件
  - CountDownLatch：倒计时器
  - CyclicBarrier：循环屏障
  - Semaphore：信号量
  - ExecutorService：线程池（FixedThreadPool、CachedThreadPool、ScheduledThreadPool）
- ThreadLocal：线程本地变量、内存泄漏问题
- 线程安全集合：ConcurrentHashMap、CopyOnWriteArrayList

### 1.3 IO/NIO
- 传统IO：InputStream/OutputStream、Reader/Writer
- NIO三大核心：Channel、Buffer、Selector
- BIO vs NIO vs AIO区别
- 文件操作：File、Path、Files工具类

### 1.4 反射
- Class对象获取方式
- 获取构造器、方法、字段
- 动态调用：invoke()
- 注解处理：反射获取注解信息

### 1.5 JVM内存模型与GC
- JVM内存结构：堆、栈、方法区、程序计数器、本地方法栈
- 对象创建过程：分配内存、初始化、设置对象头
- 垃圾回收算法：标记-清除、标记-复制、标记-整理
- GC收集器：Serial、Parallel、CMS、G1、ZGC
- 内存泄漏与内存溢出场景及排查
- JVM参数调优：堆大小(-Xms/-Xmx)、年轻代(-Xmn)、GC日志(-XX:+PrintGCDetails)

### 1.6 集合源码
- List：ArrayList底层实现（数组扩容）、LinkedList（双向链表）、Vector
- Map：HashMap底层实现（数组+链表+红黑树）、HashTable、TreeMap
- Set：HashSet、TreeSet

### 1.7 异常处理
- Exception分类：Checked Exception、Runtime Exception
- 异常处理最佳实践：不要吞异常、早抛出晚捕获、自定义异常

## 二、后端框架

### 2.1 Spring框架
- IOC容器：Bean生命周期、依赖注入方式（构造器注入、setter注入、字段注入）
- AOP：切面编程、JDK动态代理vsCGLIB代理、@Aspect注解使用
- Spring事务：@Transactional注解、事务传播机制、事务隔离级别、回滚策略
- Spring核心注解：@Autowired、@Resource、@Component、@Service、@Repository、@Controller

### 2.2 SpringMVC
- 请求处理流程：DispatcherServlet、HandlerMapping、HandlerAdapter、ViewResolver
- 常用注解：@Controller、@RequestMapping、@GetMapping、@PostMapping、@RequestBody、@ResponseBody、@PathVariable、@RequestParam
- 参数绑定与校验：@Valid、@Validated、自定义校验器
- 文件上传下载

### 2.3 SpringBoot
- 自动配置原理：@SpringBootApplication、@EnableAutoConfiguration、自动配置类
- 启动流程：SpringApplication.run()执行过程
- 配置文件：application.yml/properties配置优先级、profile多环境配置
- 常见starter：spring-boot-starter-web、spring-boot-starter-jdbc、spring-boot-starter-data-redis
- 热部署：spring-boot-devtools

### 2.4 MyBatis
- 核心组件：SqlSessionFactory、SqlSession、Mapper接口、Executor
- 配置文件：mybatis-config.xml、Mapper.xml映射文件
- 动态SQL：if、choose、foreach、where、set
- 缓存机制：一级缓存（SqlSession级别）、二级缓存（Mapper级别）
- 分页插件：PageHelper
- 注解方式：@Select、@Insert、@Update、@Delete

### 2.5 Hibernate（了解）
- ORM概念、SessionFactory、Session
- HQL查询、Criteria查询

### 2.6 SpringCloud（了解）
- Eureka/Nacos服务注册与发现
- Ribbon/Feign服务调用
- Hystrix/Sentinel熔断降级
- Gateway网关
- Config配置中心

### 2.7 Spring WebFlux响应式编程
- 响应式编程概念：Reactive Streams规范、背压（Backpressure）
- WebFlux核心组件：Router Functions、WebHandler、HandlerAdapter
- Reactor框架：Mono（0/1元素）、Flux（0-N元素）
- 常用操作符：map、flatMap、filter、reduce、zip、concat
- 响应式数据源：R2DBC、MongoDB Reactive、Redis Reactive
- WebFlux vs SpringMVC对比：适用场景、性能差异
- @Controller响应式：@GetMapping返回Mono/Flux、@ResponseBody

### 2.8 规则引擎与批处理
- QLExpress：规则表达式引擎、业务规则动态配置
- SpringBatch：批处理框架、Step、Job、ItemReader/ItemProcessor/ItemWriter
- Quartz：定时任务框架、Cron表达式、Job与Trigger

## 三、数据库与中间件

### 3.1 MySQL
- 存储引擎：InnoDB vs MyISAM区别
- InnoDB特性：事务支持、MVCC、行锁、聚簇索引
- 索引原理：B+树结构、索引类型（主键索引、唯一索引、普通索引、组合索引）
- 索引优化：覆盖索引、最左前缀原则、避免索引失效场景
- 慢SQL优化：explain分析执行计划、优化思路
- 事务隔离级别：读未提交、读已提交、可重复读、串行化
- 锁机制：行锁、表锁、意向锁
- 分库分表：垂直分表、水平分表、中间件（ShardingSphere）
- 主从复制：原理、读写分离

### 3.2 Oracle
- 表空间、用户、角色管理
- 索引类型：B树索引、位图索引、函数索引
- PL/SQL：存储过程、触发器、游标
- 事务控制：COMMIT、ROLLBACK、SAVEPOINT
- 数据库连接池：Oracle Connection Pool

### 3.3 Oracle转MySQL迁移
- 数据类型映射：NUMBER→DECIMAL/INT、VARCHAR2→VARCHAR、DATE→DATETIME
- SQL语法差异：NVL→IFNULL、SUBSTR→SUBSTRING、ROWNUM→LIMIT、CONNECT BY→递归查询
- 存储过程/触发器迁移方案
- 数据迁移工具：Kettle、DataX、mysqldump

### 3.4 Redis
- 数据结构：String、Hash、List、Set、ZSet、Bitmap、HyperLogLog、GEO
- 持久化：RDB、AOF、混合持久化
- 缓存策略：缓存穿透、缓存击穿、缓存雪崩及解决方案
- 分布式锁：基于Redis实现分布式锁
- 主从复制、哨兵模式、Cluster集群
- Redis事务：MULTI、EXEC、DISCARD、WATCH

### 3.5 MongoDB
- 文档存储、BSON格式（Binary JSON）
- 数据库与集合：MongoDB中的database、collection、document
- 索引：单字段索引、复合索引、文本索引、地理空间索引、TTL索引
- 查询操作：find、findOne、投影查询、排序、分页（skip/limit）
- 聚合框架：aggregate聚合管道、$match、$group、$project、$lookup、$sort、$limit
- 更新操作：update、updateOne、updateMany、$set、$inc、$push、$pull
- 删除操作：deleteOne、deleteMany
- 事务支持：4.0+版本支持多文档事务
- 分片集群：shard key选择、分片策略、mongos路由、config server
- 副本集：主从复制、自动故障转移、选举机制
- 与MySQL对比：适用场景、事务支持、查询灵活性

### 3.6 RabbitMQ
- AMQP协议、Exchange类型（Direct、Fanout、Topic、Headers）
- 消息可靠性：生产者确认（publisher confirms）、消费者确认（ack）、死信队列
- 消息持久化：Exchange持久化、Queue持久化、Message持久化
- 消息积压处理、消息幂等性
- 集群与镜像队列

### 3.7 服务通信
- CXF WebService：SOAP协议、WSDL、JAX-WS
- RESTful接口：HTTP方法、状态码、接口设计规范
- RPC框架：Dubbo、gRPC

## 四、分布式系统与运维工程化

### 4.3 CI/CD

### 4.4 容器云
- Docker：镜像、容器、Dockerfile编写、Docker Compose

### 4.5 服务器运维

- Weblogic：部署、配置、数据源管理
- Nginx：反向代理、负载均衡、静态资源服务、配置优化
