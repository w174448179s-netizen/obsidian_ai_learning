# 阿里云百炼

## 一句话描述
阿里云的大模型服务平台，提供通义千问等模型的 API 调用。

## 核心概念
- **API Key 管理**：AccessKey 和 SecretKey 认证
- **模型广场**：丰富的模型选择（Qwen系列、DeepSeek等）
- **DashScope API**：统一的模型调用接口

## 接入方式
- 通过 Spring AI 的 `spring-ai-alibaba-starter`
- 直接 HTTP 调用 DashScope API

## 代码示例

### Spring AI 接入
```java
@Bean
ChatClient chatClient() {
    return new ChatClient.Builder(
        new AlibabaChatModel(alibabaApiKey)
    ).build();
}
```

### DashScope API 接入
```java
// 通过 DashScope SDK 直接调用
DashScopeClient client = new DashScopeClient(apiKey);

CompletionResponse response = client.call(
    CompletionRequest.builder()
        .model("qwen-plus")
        .prompt("Hello")
        .build()
);
```

## 常见坑点
- API Key 需要妥善保管，不要暴露到前端
- 不同模型的参数范围可能不同
- 需要注意模型的 Token 限制和计费规则
