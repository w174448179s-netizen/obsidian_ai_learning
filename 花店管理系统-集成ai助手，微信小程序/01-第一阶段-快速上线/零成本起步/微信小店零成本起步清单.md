# 微信小店零成本起步清单

> 目标：**0 平台费**验证线上卖花，先跑通"能卖出去"再谈花钱
> 预计耗时：开店 3-7 天，商品录入 + 上线 1-2 周
> 平台费用：0 元（仅 0.6% 交易手续费，新商家可 0 保证金试运营）

## 前置准备

### 资质

- [ ] 办个体工商户营业执照（经营范围含鲜花/礼品零售，当地工商或线上免费办）
- [ ] 法人身份证正反面照片（清晰、不反光）
- [ ] 法人本人储蓄银行卡（用于收款结算）

### 商品素材（最花时间，提前备好再开店）

- [ ] 花束实拍图（统一背景与光线，建议白底或浅色，每款 3-5 张）
- [ ] 花束名称、规格（朵数 / 尺寸 / 包装）
- [ ] 价格表（含成本价，用于核算利润与定价）
- [ ] 花语文案（每种花材的花语、适合送礼场景）
- [ ] 商品分类规划（玫瑰 / 百合 / 康乃馨 / 混搭 / 绿植 / 花篮）

### 运营资料

- [ ] 花店门头照片、店内环境照
- [ ] 配送范围说明（按距离或区域）
- [ ] 配送时段与起送价
- [ ] 客服联系方式、营业时间

## 开店入驻

- [ ] 微信搜索"微信小店开店"或访问 store.weixin.qq.com
- [ ] 选择"个体工商户店"
- [ ] 上传营业执照（系统自动识别，核对信息）
- [ ] 填写法人信息 + 绑定银行卡
- [ ] 店铺命名（好记、无"最/第一"等违规词）
- [ ] 管理员人脸识别验证（需本人操作）
- [ ] 提交审核（1-3 个工作日）
- [ ] 审核通过，进入店铺后台（"微信小店助手"小程序 + PC 端 shop.weixin.qq.com）

## 店铺搭建（0 代码装修）

### 基础设置

- [ ] 上传店铺头像、店招
- [ ] 填写店铺简介（花店故事、特色）
- [ ] 设置客服（自动回复 + 人工接待）
- [ ] 设置营业时间

### 商品录入

- [ ] 创建商品分类
- [ ] 逐个录入商品（图片、名称、价格、规格、库存、花语描述）
- [ ] 设置商品规格（尺寸 / 朵数 / 包装选项）
- [ ] 设置库存数量
- [ ] 上架商品

### 配送与支付

- [ ] 确认微信支付已开通（小店自带，0.6% 费率）
- [ ] 设置配送范围与运费模板
- [ ] 设置配送时段
- [ ] 设置自提选项（如有门店）

## 上线验证

- [ ] 自己走完整下单流程（浏览 → 下单 → 支付 → 确认 → 退款）
- [ ] 检查商品图片、价格、规格显示正常
- [ ] 测试客服消息接收
- [ ] 生成店铺 / 商品分享链接
- [ ] 把小店挂到视频号主页
- [ ] 朋友圈 / 社群首次推广

## 上线后运营观察（1-2 个月）

每周记录以下数据，作为是否升级的依据：

- [ ] 日访客数、日单量
- [ ] 客单价
- [ ] 热销款、滞销款
- [ ] 客户反馈与复购情况
- [ ] 节日节点备货与促销

## 什么时候花钱升级（触发条件）

满足任一即可考虑投入：

- 月订单稳定增长，微信小店模板/营销限制了你（想要拼团、秒杀、分销等玩法）
- 想要自己的品牌小程序门面 → 花 300 元/年做小程序认证 + 嵌入小店，仍 0 平台费
- 想要更深度会员体系与数据分析 → 上 SaaS（约 700 元/年）或直接自研

**升级路径**：微信小店(0 元) → 小程序壳嵌小店(300 元/年) → SaaS 模板(700 元/年) → 自研开发

## 关键提醒

- 先备好商品素材再开店，开店后能快速上架、快速验证
- 微信小店模板样式固定，若对视觉要求高，后期再上 SaaS
- 数据定期导出备份，为后续自研迁移做准备
- 第一阶段核心是"验证能卖出去"，不必追求系统完美
