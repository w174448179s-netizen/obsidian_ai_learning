# SaaS上线操作清单

> 预计耗时：1-2周

## 前置准备

### 微信生态注册

- [ ] 注册微信公众平台账号（mp.weixin.qq.com）
- [ ] 选择"小程序"类型注册
- [ ] 完成小程序认证（300元/年，需营业执照/个体工商户）
- [ ] 注册微信支付商户号（pay.weixin.qq.com）
- [ ] 将商户号绑定到小程序

### 资料准备

- [ ] 营业执照照片
- [ ] 法人身份证正反面
- [ ] 对公账户信息（微信支付需要）
- [ ] 花店门头照片
- [ ] 花束商品图片（建议统一风格、白底/浅色背景）
- [ ] 花束价格表
- [ ] 配送范围和规则说明
- [ ] 花店联系方式

## SaaS平台搭建

### 平台购买与绑定

- [ ] 注册乔拓云账号
- [ ] 购买标准版套餐
- [ ] 授权绑定微信小程序
- [ ] 配置微信支付参数

### 商品录入

- [ ] 创建商品分类（玫瑰/百合/康乃馨/混搭花束/绿植/花篮等）
- [ ] 录入商品信息（名称、图片、价格、描述、花语）
- [ ] 设置商品规格（尺寸、包装、花材数量）
- [ ] 设置库存数量
- [ ] 设置商品状态（上架/下架）

### 配送配置

- [ ] 设置配送范围（地图圈选或按区域）
- [ ] 设置配送时间段
- [ ] 设置配送费用规则
- [ ] 设置自提点信息（如有）
- [ ] 设置节日特殊配送规则

### 支付与订单

- [ ] 配置微信支付
- [ ] 设置订单自动取消时间
- [ ] 配置订单通知（短信/模板消息）

## 测试验证

- [ ] 完整下单流程测试（浏览→选品→下单→支付→确认）
- [ ] 退款流程测试
- [ ] 配送范围边界测试
- [ ] 不同规格价格计算验证
- [ ] 会员功能测试

## 发布上线

- [ ] 检查所有页面展示正常
- [ ] 检查所有链接可用
- [ ] 提交微信审核
- [ ] 审核通过后发布上线
- [ ] 生成小程序码，张贴在花店内
- [ ] 通知老客户试用

## 上线后运营

- [ ] 每日检查订单处理
- [ ] 定期更新商品图片和价格
- [ ] 收集用户反馈
- [ ] 记录运营数据（日单量、客单价、复购率）
- [ ] 评估是否进入第二阶段自研
