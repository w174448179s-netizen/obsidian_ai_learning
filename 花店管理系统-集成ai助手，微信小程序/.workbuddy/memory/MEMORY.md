# 花店小程序项目 - 长期记忆

## 项目定位
花店微信小程序，三阶段推进：SaaS快速上线 → 自研开发 → AI集成。项目文档在 `00-项目总览/花店微信小程序三阶段实施计划.md`。

## 关键决策与纠正

### 起步成本策略（2026-07-19 纠正）
- **微信小店是零成本起点**：开店免费，仅0.6%交易手续费，新商家0保证金试运营。原计划直接上SaaS(700元/年)成本偏高。
- 修正后成本阶梯：
  - 0级（0元）：微信小店 + 视频号/朋友圈，先验证能卖出去
  - 1级（0~300元/年）：自己的小程序壳 + 嵌入微信小店
  - 2级（~700元/年）：SaaS模板（乔拓云），卖动后升级
  - 3级：自研开发
- 主体建议：个体工商户执照（免费办），纯自然人入驻不适合鲜花实体销售
- 待办：用户尚未确认是否更新第一阶段文档为"微信小店先行"

## 技术选型（已定）
- 第二阶段自研：uni-app(Vue3) + Node.js + MySQL + Redis + 对象存储
- 第三阶段AI：大模型API + RAG，候选模型 DeepSeek/Qwen/豆包/GLM-4
