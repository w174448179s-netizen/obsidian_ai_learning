将新学习的概念增量添加到思维导图中。

## 执行步骤
1. 分析对话历史，识别新学习的概念和知识点
2. 读取现有思维导图文件，了解当前结构
3. 将新概念添加到对应分支下，保持层级清晰
4. 使用editFile工具直接更新思维导图文件

## 思维导图结构规范

```markdown
- AI应用开发学习
  - 核心处理管道
    - 1-接入网关
      - 子概念1
      - 子概念2
  - 共享上下文层
    - 7-记忆管理
      - 子概念1
  - 全链路横切面
    - 8-审计日志
  - 技术栈
    - Spring AI
      - 子概念1
  - 学习资源
  - 版本迭代
```

## 更新规则
1. **增量添加**：只添加新节点，不删除已有节点
2. **层级匹配**：新概念放在对应模块的分支下
3. **命名规范**：使用简短清晰的概念名称
4. **避免重复**：检查是否已存在相同概念

## 输出格式
### 📊 思维导图更新结果
- 更新文件：[[AI应用开发学习]]
- 添加节点：
  - 核心处理管道 > 1-接入网关 > 新概念1
  - 技术栈 > Spring AI > 新概念2

### 更新摘要
```markdown
    - 1-接入网关
      - 新概念1
      - 新概念2
```

## 使用方式
用户完成某个概念学习后，点击此提示词，AI会自动分析对话内容并更新思维导图。
